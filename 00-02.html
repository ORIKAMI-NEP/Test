プログラミング言語

プログラミング言語とは、簡単に言えば コンピュータが理解できる言語 です。

「"こんにちは"という文を出力」←これは私たち日本人には理解できますが、コンピュータには理解できません。英語に直しても同じです。そこでコンピュータにも理解できるように作られた言語がプログラミング言語なのです。

世界にはC言語やJava、Python、PHPなど200種類以上もの言語が存在しています。

Javaの説明

今回はその数あるプログラミング言語の中から、Javaという言語について説明したいと思います。

Javaは世界的に見ても使用されている言語、習得したい言語として1位を記録するほどに実績のある言語です。なぜここまで人気が高いのかというと、
「Javaは多くの現場で使われている」 、 「汎用的でOSに依存しない」
からなんです。

OSに依存しないという点はとても重要で、例えばWindows用に作ったプログラムをMacでも動かしたい時、他の大半の言語であればわざわざMac用に書き換えなければいけません。しかしJavaはその必要がないのです。

↓Javaのコード例 

                  
class hoge {
    public static void main(String args[]) {
        System.out.println(s@"Hello World!"@s)
    }
}

なぜJavaから勉強するのか

さて、プログラミング同好会ではなぜJavaを初めに勉強するのか、それはずばり
情報コースで1番初めに勉強するから
です。なんやねんと思うかもしれませんが、プログラミング言語は一度に複数を勉強するのは難しいと言われています。ただでさえ意味不明な文章のルールを、別の言語で分けて覚えようものなら頭がパンクするでしょう（多分）。よって、2年生で勉強するときにスムーズに理解できるように、まずはJavaを学んでもらいます。
